COG 지도 예제

같은 COG GeoTIFF와 탐지 GeoJSON을 OpenLayers와 Leaflet에서 각각 불러오는 GitHub Pages용 정적 예제입니다.

OpenLayers COG + GeoJSON GeoTIFF source로 COG를 직접 타일처럼 표시하고 탐지 GeoJSON을 겹칩니다. Leaflet COG + GeoJSON georaster-layer-for-leaflet으로 COG를 표시하고 Leaflet 레이어 컨트롤을 붙입니다. GeoServer OpenLayers WMS GeoServer의 water workspace WMS 레이어 5개를 선택해 시각화합니다.

파일

  • leaflet_BSG-119_4326_preview_cog.tif
  • BSG-119-20260406-224609-435441813_ortho_detections_v2.geojson
  • detections.cleaned.geojson for the repaired BSG-131 GeoServer overlay

GitHub Pages에 올릴 때 1GB 이상의 GeoTIFF는 일반 GitHub 파일 제한에 걸릴 수 있습니다. 필요하면 COG만 별도 객체 스토리지나 릴리즈 자산에 두고 각 HTML의 COG_URL만 바꾸면 됩니다. OpenLayers와 Leaflet 예제는 GitHub Pages 배포를 위해 EPSG:4326 경량 COG를 사용합니다. BSG-131 원본 탐지 GeoJSON은 일부 문자열과 마지막 feature가 깨져 있어, GeoServer 예제에서는 geometry와 핵심 속성만 추출한 정리본을 사용합니다.
